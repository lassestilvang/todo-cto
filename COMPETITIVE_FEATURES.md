### Quick Workflows

**Morning Planning**:
1. Open Analytics (sidebar) to review yesterday
2. Check "Today" view
3. Use Focus Mode (Ctrl+Shift+F) for top 3 tasks
4. Start Pomodoro (Ctrl+Shift+P) for each task

**Rapid Task Capture**:
1. Press Ctrl+Q anywhere
2. Type naturally: "Team meeting tomorrow 1h urgent"
3. Press Enter - done!

**Deep Work Session**:
1. Navigate to Today view
2. Click Focus Mode button (or Ctrl+Shift+F)
3. Press Space to mark complete, arrows to navigate
4. Press P to start Pomodoro timer
